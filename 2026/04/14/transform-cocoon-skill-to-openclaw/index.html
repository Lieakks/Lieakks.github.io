改造 Cocoon AI Skill 为 OpenClaw 可用

发表于 2026-04-14 分类于 技术

原因

Cocoon AI 的 architecture-diagram-generator 是 Claude.ai 专用 Skill，OpenClaw 需要 YAML frontmatter 才能识别加载。

方法

改动 原始 改造后
加 frontmatter 无 name + description + metadata.openclaw
移模板 template.html assets/template.html
保设计 - 沿用暗色配色系统

文件结构：

效果

现已可用：描述系统架构 → 生成独立 HTML 文件（内嵌 SVG），浏览器直接打开查看。
