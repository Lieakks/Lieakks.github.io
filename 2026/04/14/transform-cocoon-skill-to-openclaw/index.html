基于 OpenClaw 改造 Cocoon AI 暗色架构图生成器 Skill

发表于 2026-04-14 分类于 技术

我把 Cocoon AI 的 architecture-diagram-generator 改造成了可直接在 OpenClaw 调用的 skill：输入架构描述 → 输出暗色主题、独立 HTML（内嵌 SVG）的架构图。

原工具

  • Cocoon AI：architecture-diagram-generator
  • 链接：https://github.com/Cocoon-AI/architecture-diagram-generator

为什么要改造它（我用 OpenClaw 的原因）

原工具是 Claude.ai 专用交互/加载方式；而 OpenClaw 对 skill 有固定规范：SKILL.md 必须有 YAML frontmatter，且资源文件需要按约定放在 skill 目录下才能被正确读取。

改造过程（基于经验的最小可行改动）

  1. 补齐 YAML frontmatter：为 OpenClaw 声明 name/description/metadata.openclaw。
  2. 调整模板资源路径：把模板挪到 assets/template.html，并确保 SKILL.md 引用到正确位置。
  3. 保持原设计系统：保留暗色配色、SVG 样式与布局规则（只做 OpenClaw 兼容层改动）。

结果（可验证、可复用）

改造后，OpenClaw 已能正确加载并调用该 skill：

用法：描述系统架构 → 生成独立 HTML 文件（内嵌 SVG），直接在浏览器打开查看/分享。
