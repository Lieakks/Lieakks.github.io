基于 OpenClaw 改造 Cocoon AI 架构图生成 Skill

发表于 2026-04-14 分类于 技术笔记

Cocoon AI 开源了一款暗色主题架构图生成器（architecture-diagram-generator），可将自然语言描述的系统架构直接渲染为独立 HTML 文件（内嵌 SVG），无需任何绘图工具。原版为 Claude.ai 专属，本文记录将其迁移到 OpenClaw 环境的过程与结论。

1. 原工具介绍

项目 内容
名称 architecture-diagram-generator
作者 Cocoon AI
功能 自然语言 → 暗色主题 SVG 架构图（独立 HTML）
设计亮点 组件分色系统（前端/后端/数据库/云服务）、JetBrains Mono 字体、z-order 箭头遮挡处理

2. 改造动机

OpenClaw 对 skill 有固定的加载规范：

  1. SKILL.md 必须包含 YAML frontmatter（name、description、metadata.openclaw），否则无法被识别和加载
  2. 资源文件需放在 skill 目录下的 assets/ 子目录，供运行时正确读取

原工具的 SKILL.md 不含 frontmatter，模板路径也不符合 OpenClaw 约定，因此需要最小化改造。
